# WeatherDetector

WeatherDetector is a simple Python-based command-line program that displays a simulated weather report for any city entered by the user. It demonstrates basic Python concepts such as functions, modules, random data generation, user input handling, and clean project structure.

## Features
- Enter any city name to receive a mock weather report
- Randomized temperature, humidity, and conditions
- Clean code structure with separate modules
  
## Technologies Used
- Python 3
- Random module
- Modular programming (separate files)
